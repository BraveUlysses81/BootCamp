1. HTML
2. Linux
3. CSS
4. CSS / JavaScript
5. JavaScript
6. Bootstrap
7. Database / SQL
8. Express
9. Express Routing / MongoDB
10. AJAX / NPM importing of packages
11. EJS / consuming API's
12. EJS partials/ AJAX to-do app
13. React basics
14. React components
15. React sign-up and CSS
